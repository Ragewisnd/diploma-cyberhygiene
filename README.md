# diploma-cyberhygiene

Локальная система обучения пользователей основам кибергигиены с входным и итоговым тестированием.

## Стек
- Backend: FastAPI
- Frontend: React
- Database: MongoDB
- Containers: Docker Compose
- MongoDB connection string: mongodb://localhost:27017/cyberhygiene

## Структура проекта
- backend/ - серверная часть
- frontend/ - клиентская часть
- docs/ - документация и материалы диплома

## Запуск
Проект разворачивается локально на macOS с использованием Docker Compose.
